# Brain Games Collection - 게임 컨셉 문서

## 📋 게임 개요

### 게임명
**Brain Games Collection**

### 장르
두뇌 훈련 / 퍼즐 / 아케이드

### 플랫폼
모바일 (iOS, Android)

### 타겟 유저
- **주 타겟**: 자기 계발과 두뇌 회전에 관심 있는 20-40대
- **부 타겟**: 치매 예방을 원하는 시니어 층, 교육용 게임을 찾는 학부모
- **게임 성향**: 짧은 시간에 집중해서 즐길 수 있는 캐주얼 게임 선호 유저

### 핵심 컨셉
"매일 10분, 당신의 두뇌를 깨우는 습관"
직관적이고 세련된 디자인의 미니게임들을 통해 기억력, 순발력, 계산력, 논리력을 즐겁게 훈련합니다.

## 🎮 게임 모드

### 1. 🎴 Flip & Match (기억력)
카드의 위치를 기억하고 짝을 맞추는 클래식 메모리 게임입니다.
- **규칙**: 뒤집힌 카드의 짝을 찾아 매칭합니다.
- **난이도**:
  - Easy: 4x4 (8쌍)
  - Medium: 4x6 (12쌍)
  - Hard: 4x8 (16쌍)
- **점수 산정**: 소요 시간과 이동 횟수를 기반으로 계산

### 2. 🔢 Sequence (순발력 & 기억력)
화면에 나타나는 숫자의 순서를 기억하고 순서대로 터치하는 게임입니다.
- **규칙**: 1부터 시작하여 증가하는 숫자를 순서대로 빠르게 터치합니다.
- **특징**: 레벨이 올라갈수록 숫자의 개수가 늘어나거나 화면이 복잡해집니다.
- **점수 산정**: 정확도와 반응 속도

### 3. ➕ Math Rush (계산력)
제한 시간 내에 간단한 사칙연산 문제를 빠르게 풀어내는 게임입니다.
- **규칙**: 화면에 제시된 수식의 정답을 4지선다에서 선택합니다.
- **특징**: 30초 제한 시간, 연속 정답 시 콤보 보너스 획득.
- **점수 산정**: 정답 수 + 콤보 보너스

### 4. 🧩 Merge Puzzle (논리력)
숫자 타일을 합쳐 더 큰 숫자를 만들어내는 2048 스타일의 퍼즐 게임입니다.
- **규칙**: 같은 숫자의 타일을 합치면 두 배의 숫자가 됩니다.
- **목표**: 제한된 이동 횟수 내에 목표 숫자(예: 64, 128) 만들기.
- **점수 산정**: 달성한 최대 숫자와 남은 이동 횟수

### 5. 🧠 Spatial Memory (공간 기억력)
잠깐 보여지는 타일의 위치를 기억해내는 게임입니다.
- **규칙**: 3x3 ~ 6x6 그리드에서 반짝인 타일의 위치를 기억했다가 터치합니다.
- **특징**: 레벨이 오를수록 기억해야 할 타일 수가 증가합니다.

### 6. 🎨 Stroop Test (인지 능력)
글자의 색상과 글자의 의미가 다를 때, 글자의 '색상'을 맞추는 게임입니다.
- **규칙**: "파랑"이라는 글자가 빨간색으로 쓰여있다면, '빨강'을 선택해야 합니다.
- **특징**: 좌뇌와 우뇌의 갈등을 극복하고 빠른 판단력을 요구합니다.

## 🌐 온라인 & 소셜 기능

### 멀티플레이어 대전
실시간으로 친구 또는 전 세계 유저와 경쟁합니다.
- **실시간 점수 동기화**: 상대방의 진행 상황을 실시간으로 확인하며 긴장감 조성.
- **매치메이킹**: 비슷한 실력의 유저와 자동 매칭.
- **이모지 채팅**: 간단한 이모지로 감정 표현.

### 리더보드
- **글로벌 랭킹**: 전 세계 유저들과 점수 경쟁.
- **친구 랭킹**: 등록된 친구들 간의 순위 경쟁.
- **주간 리그**: 매주 초기화되는 리그 시스템으로 지속적인 동기 부여.

### 친구 시스템
- 친구 추가 및 관리
- 친구의 접속 상태 확인 (온라인/오프라인)
- 친구 기록 비교 및 도전장 보내기

## 🎨 디자인 철학

### Visual Identity
- **Premium & Clean**: 불필요한 장식을 배제하고 콘텐츠에 집중할 수 있는 미니멀리즘 디자인.
- **Glassmorphism**: 반투명한 유리에 빛이 투과되는 듯한 깊이감 있는 UI.
- **Vivid Gradients**: 게임별로 고유한 그라데이션 컬러를 사용하여 시각적 즐거움 제공.

### UX 원칙
- **직관성**: 설명서 없이도 바로 플레이 가능한 UI.
- **피드백**: 터치, 성공, 실패 등 모든 상호작용에 대한 시각/청각/햅틱 피드백 제공.
- **접근성**: 다크 모드 지원, 색약 모드 고려.
