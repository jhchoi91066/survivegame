개인정보처리방침 / Privacy Policy

최종 수정일 / Last Updated: 2025년 10월 6일 / October 6, 2025
버전 / Version: 3.0.0

1. 개요 / Overview

Brain Games는 사용자의 개인정보를 중요하게 생각하며, 개인정보 보호에 최선을 다하고 있습니다.

Brain Games values user privacy and is committed to protecting personal information.

2. 수집하는 정보 / Information We Collect

2.1 로컬 저장 (오프라인) / Local Storage (Offline)

  • 게임 기록 (최고 점수, 최단 시간) / Game records (high scores, best times)
  • 게임 통계 (플레이 횟수, 총 플레이 시간) / Game statistics (play count, total play time)
  • 앱 설정 (다크/라이트 모드) / App settings (dark/light mode)

2.2 온라인 기능 (선택적) / Online Features (Optional)

로그인 선택 시에만 / Only when you choose to log in:

  • 익명 사용자 ID / Anonymous user ID
  • 사용자명 / Username
  • 게임 기록 및 순위 / Game records and rankings
  • 친구 관계 정보 / Friend relationships
중요 / Important:
  • 개인 식별 정보 수집 안 함 / No personal identification collected
  • 익명 로그인 / Anonymous login
  • 로그인 100% 선택사항 / Login is 100% optional

3. 정보의 사용 / How We Use Information

  • 게임 진행 저장 / Save game progress
  • 통계 표시 / Display statistics
  • 리더보드 순위 (로그인 시) / Leaderboard rankings (when logged in)
  • 클라우드 동기화 (로그인 시) / Cloud sync (when logged in)

4. 제3자 서비스 / Third-Party Services

  • Supabase: 백엔드 데이터베이스 / Backend database
    Privacy Policy: https://supabase.com/privacy

5. 사용자 권리 / User Rights

  • 데이터 다운로드 / Data download: 설정 > 내 데이터 다운로드 / Settings > Download My Data
  • 계정 삭제 / Account deletion: 설정 > 계정 삭제 / Settings > Delete Account

6. 아동 안전 / Children's Safety

13세 이상 사용자 대상 / For users aged 13 and above
