# 개인정보처리방침

**최종 수정일: 2025년 10월 6일**
**버전: 3.0.0**

## 1. 개요

Brain Games("본 앱")는 사용자의 개인정보를 중요하게 생각하며, 개인정보 보호에 최선을 다하고 있습니다.

## 2. 수집하는 정보

### 2.1 로컬 저장 정보 (오프라인)
본 앱은 다음의 정보를 **기기 내부에만** 저장합니다:
- 게임 기록 (최고 점수, 최단 시간 등)
- 게임 통계 (플레이 횟수, 총 플레이 시간)
- 앱 설정 (테마 설정: 다크/라이트 모드)
- 온보딩 완료 여부

### 2.2 온라인 기능 사용 시 수집 정보 (선택적)
**로그인을 선택한 경우에만** 다음 정보가 수집됩니다:
- 익명 사용자 ID (Supabase 자동 생성)
- 사용자명 (사용자 직접 설정)
- 국가 코드 (선택 사항)
- 게임 기록 및 순위
- 친구 관계 정보
- 업적 진행도

**중요:**
- 이메일, 전화번호, 실명 등 개인 식별 정보는 수집하지 않음
- 익명 로그인 사용 (이메일/비밀번호 불필요)
- 로그인은 100% 선택사항이며, 로그인 없이도 모든 게임 플레이 가능

## 3. 정보의 사용

수집된 정보는 다음 목적으로만 사용됩니다:
- 게임 진행 상황 저장
- 통계 및 기록 표시
- 사용자 설정 유지
- 온라인 리더보드 순위 표시 (로그인 시)
- 친구 시스템 운영 (로그인 시)
- 클라우드 동기화 (로그인 시)

## 4. 정보의 저장

### 4.1 로컬 저장
오프라인 데이터는 **기기 내부(AsyncStorage)**에만 저장되며:
- 외부 서버로 전송되지 않음
- 앱 삭제 시 모든 데이터가 함께 삭제됨

### 4.2 클라우드 저장 (로그인 시)
온라인 기능 사용 시 데이터는 **Supabase(PostgreSQL)**에 저장됩니다:
- 엄격한 Row Level Security (RLS) 정책 적용
- 본인의 데이터만 접근 가능
- 암호화된 연결(HTTPS)을 통해 전송
- 계정 삭제 시 모든 클라우드 데이터 영구 삭제

## 5. 제3자 서비스

### 5.1 사용하는 서비스
- **Supabase**: 온라인 기능을 위한 백엔드 서비스 (로그인, 데이터베이스)
  - 데이터 위치: 안전한 클라우드 데이터센터
  - 목적: 게임 기록 동기화, 리더보드, 친구 시스템
  - 개인정보 보호 정책: https://supabase.com/privacy

### 5.2 사용하지 않는 서비스
본 앱은 다음의 제3자 서비스를 사용하지 않습니다:
- 광고 네트워크
- 분석 서비스 (Analytics)
- 소셜 미디어 통합
- 위치 추적 서비스

## 6. 권한

본 앱이 요청하는 권한:
- **진동 (VIBRATE)**: 게임 피드백을 위한 햅틱 진동에만 사용

## 7. 아동의 개인정보 보호

본 앱은 모든 연령층이 안전하게 사용할 수 있습니다. 어떠한 개인정보도 수집하지 않으므로 아동이 사용하기에 안전합니다.

## 8. 보안

모든 데이터는 기기 내부에 암호화되어 저장되며, 네트워크를 통해 전송되지 않습니다.

## 9. 사용자 권리

### 9.1 데이터 접근 및 관리
사용자는 다음 권리를 가집니다:
- 본인의 데이터 조회 (프로필, 게임 기록, 친구 목록)
- 온라인 설정 변경 (자동 동기화, 리더보드 참여, 친구 요청 수신)
- 언제든지 로그아웃 가능

### 9.2 데이터 삭제
사용자는 언제든지 다음 방법으로 데이터를 삭제할 수 있습니다:
1. **로컬 데이터**: 앱 내 설정 > "진행도 초기화"
2. **온라인 계정**: 앱 내 설정 > 계정 관리 > "계정 삭제"
   - 모든 클라우드 데이터 영구 삭제
   - 친구 관계, 리더보드 기록 모두 삭제
   - 복구 불가능
3. **앱 삭제**: 기기의 모든 로컬 데이터 삭제

## 10. 정책 변경

개인정보처리방침이 변경될 경우, 앱 업데이트를 통해 공지됩니다.

# Privacy Policy (English)

**Last Updated: October 6, 2025**
**Version: 3.0.0**

## 1. Overview

Brain Games ("the App") values your privacy and is committed to protecting your personal information.

## 2. Information We Collect

### 2.1 Local Storage Information (Offline)
The App stores the following information **locally on your device only**:
- Game records (high scores, best times, etc.)
- Game statistics (play count, total play time)
- App settings (theme: dark/light mode)
- Onboarding completion status

### 2.2 Online Features Information (Optional)
**Only when you choose to log in**, the following information is collected:
- Anonymous user ID (automatically generated by Supabase)
- Username (set by you)
- Country code (optional)
- Game records and rankings
- Friend relationships
- Achievement progress

**Important:**
- NO personal identification info (email, phone, real name)
- Anonymous login (no email/password required)
- Login is 100% optional - all games playable without logging in

## 3. Use of Information

Collected information is used only for:
- Saving game progress
- Displaying statistics and records
- Maintaining user preferences
- Online leaderboard rankings (when logged in)
- Friend system operation (when logged in)
- Cloud synchronization (when logged in)

## 4. Data Storage

All data is stored **locally on your device (AsyncStorage)** only:
- Not transmitted to external servers
- Not shared with third parties
- Deleted when the app is uninstalled

## 5. Third-Party Services

The App does NOT use:
- Advertising networks
- Analytics services
- Social media integration
- Cloud storage

## 6. Permissions

Permissions requested by the App:
- **VIBRATE**: Used only for haptic feedback during gameplay

## 7. Children's Privacy

The App is safe for all ages. Since we collect no personal information, it is safe for children to use.

## 8. Security

All data is encrypted and stored locally on your device, never transmitted over the network.

## 9. Data Deletion

Users can delete their data at any time by:
1. App Settings > Statistics > "Reset Statistics" button
2. Uninstalling the app (all data is deleted)

## 10. Policy Changes

If our privacy policy changes, we will notify users through app updates.
